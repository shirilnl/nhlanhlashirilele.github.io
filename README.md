# NLS Engineering Portfolio

Professional portfolio website for Nhlanhla Lucky Shirilele.

---

## Overview

This is a modern, responsive engineering portfolio website built with:

- HTML5
- CSS3
- JavaScript
- Progressive Web App technologies
- Service Workers
- QR Code integration
- Local analytics
- AI portfolio assistant
- Local administration tools

---

## Features

### Portfolio

- Professional profile
- Engineering experience
- Skills
- Technologies
- Projects
- Certificates
- Gallery
- Contact information

### Contact

- Phone contact
- Email contact
- WhatsApp contact
- Contact form
- QR codes
- Digital contact card

### Advanced Features

- AI portfolio assistant
- Local admin dashboard
- Portfolio data export/import
- Visitor analytics
- Project interaction tracking
- Certificate interaction tracking
- PWA support
- Offline functionality
